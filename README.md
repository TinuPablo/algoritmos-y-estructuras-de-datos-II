TRABAJO PRACTICO 1
Creamos un sistema de vehículos usando herencia y polimorfismo, haciendo una clase base (Vehiculo) y subclases (Auto y Moto) que sobrescriben el método describir().

TRABAJO PRACTICO 2
Creamos figuras geométricas, haciendo una clase base (Figura) y subclases (Rectangulo y Circulo) que sobrescriben el método area(), aplicando polimorfismo al describir cada figura. Tambien se importa el modulo "math", para usar el valor completo de pi al momento de sacar el area() del circulo, y no poner 3,14.


Intrucciones para ejecutra los archivos:
-Tener Python instalado en la computadora.
-Guardar el archivo con extensión .py (por ejemplo: ejercicio1.py o ejercicio2.py).
-En caso de no tener VSA, puede abrir la terminal.
-Ubicarse en la carpeta donde está el archivo.
-Ejecutar el programa con el comando: python nombre_del_archivo.py
